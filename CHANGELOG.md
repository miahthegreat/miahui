# Changelog

All notable changes to Miahui will be documented in this file.

## [0.1.0] - 2024-01-XX

### Added
- Initial release of Miahui component library
- CLI tool with `init`, `add`, and `list` commands
- Support for three styling approaches:
  - Tailwind CSS (with automatic CSS variable injection)
  - Vanilla CSS
  - SCSS
- Component templates:
  - Button (with variants and sizes)
  - Card (with header, content, footer)
  - Input
  - Dialog
  - Label
- Configuration system with `components.json`
- Component transformation based on style preference
- Automatic CSS variable injection for Tailwind mode
- TypeScript support
- Comprehensive documentation

### Features
- Copy-based installation (components are copied, not installed)
- Style-aware component transformation
- CSS custom properties for theming
- Full TypeScript support
- Easy customization (modify components directly)
